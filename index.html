Course Overview

This study unit focuses on software and information system projects from a wider project-management perspective. The course connects feasibility analysis, leadership, documentation, and lifecycle thinking with the practical work of building tested software.

Prerequisites Design of Information Systems, Programming II, Data Management and Databases (SQL)
Programmes Bachelor and Master level programmes in Computer Science, Information Technology, Informatics, minor and additional studies

Practical Focus

  • Managing project work from project order to tested software
  • Working with documentation throughout the full project life cycle
  • Understanding large-scale and multi-project environments
  • Combining technical work with leadership and communication

Learning Outcomes

  • Define project requirements in more detail from the perspective of feasibility analysis
  • Understand the specifics of managing large-scale project entities
  • Apply appropriate leadership practices throughout the project life cycle
  • Manage project documentation effectively

Soft Skills

  • Digitalization
  • Leadership and development
  • Critical thinking
  • Identifying and developing your own expertise
  • Communication skills
  • Social skills

Course Content

  • Feasibility analysis and its application to software and information system projects
  • Extensive projects and working in a multi-project organization
  • Leadership practices during the life cycle of the project
  • Documentation in projects
  • Running a project from the beginning, from project order until the tested software
